From .NET framework, to .NET Core, to .NET
正如右图所示，.NET核心系统由三个主要部分组成：.NET framework, .NET, 以及Xamarin。
其中Xamarin无需多说，当你想要C#构建一个移动应用时，它是你的唯一选择。
我们主要来聊聊 .NET framework, .NET core, .NET之间的差异。
.NET Framework Overview
it is released in 2002 together with C#
mainly used to create Windows desktop(WPF, Windows forms) and server-based applications(ASP.NET) using VB.NET, C# and F#
available only on WINDOWS - .NET framework在底层设计上就与WINDOWS平台死死绑定在了一起，因此不支持跨平台。
最新版本的.NET framework与win10系统一起被安装，并且会一直维护到win10的维护期限
如微软官网所说，v4.8 就是 .NET Framework 的最后一个版本，此后除了一些小的维护外不会再有大版本的更新。
What's new in .NET Framework - .NET Framework
Note .NET Framework 4.8 is the last version of .NET Framework. .NET Framework is serviced monthly with security and reliability bug fixes. .NET Framework will continue to be included with Windows, with no plans to remove it. You don't need to migrate your .NET Framework apps, but for new development, use .NET 5 or later.
https://docs.microsoft.com/en-us/dotnet/framework/whats-new/
