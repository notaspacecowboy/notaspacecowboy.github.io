ContextMenu
The ContextMenu attribute allows you to add commands to the context menu.
In the inspector of the attached script. When the user selects the context menu, the function will be executed.This is most useful for automatically setting up Scene data from the script. The function has to be non-static.
Code Sample
public class Test : MonoBehaviour { /// <summary> /// Add a context menu named "Do Something" in the inspector /// 0 is the priority of the function in the context menu /// </summary> [ContextMenu("Do Something", false, 0)] public void DoSomething() { Debug.Log("Perform operation"); } }

ContextMenuItem
Use this attribute to add a context menu to a field that calls a named method.
Code Sample
public class Test : MonoBehaviour { [ContextMenuItem("Reset", "ResetName")] public string name = ""; void ResetName() { Debug.Log("ResetName"); name = ""; } }
